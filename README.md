# Mutual Fund Analytics Dashboard

## Project Overview
This project is a Power BI dashboard developed to analyze mutual fund investment data. It provides insights into investments based on gender, transaction type, city tier, state, age group, and monthly transaction trends.

## Features
- Total Investment KPI
- Investment by Gender
- Investment by Transaction Type
- Investment by City Tier
- Investment by State
- Average SIP Amount by Age Group
- Monthly Transaction Volume
- Interactive Filters (Gender, Transaction Type)

## Tools & Technologies
- Power BI
- SQL
- Python
- SQLite
- VS Code

## Dataset
The dataset contains mutual fund investor transaction records including:
- Investor Details
- Gender
- Age Group
- State
- Transaction Type
- Investment Amount
- City Tier
- Transaction Date

## Dashboard Pages

### Page 1 – Investment Overview
- Total Investment
- Investment by Gender
- Investment by Transaction Type
- Investment by City Tier
- NAV Trend by Month

### Page 2 – Investment Analysis
- Investment by State
- Average SIP Amount by Age Group
- Monthly Transaction Volume

### Page 3 – Transaction & Demographic Analysis
- Transaction Type Split
- Investment by Gender
- Investment by State
- Average SIP Amount by Age Group
- Monthly Transaction Volume
